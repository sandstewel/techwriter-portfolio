The User Object

The User object Field Type Required Description id string Yes Unique identifier for the user. name string Yes Full name of the user. email string Yes Email address of the user. created_at string No Timestamp when the user was created (ISO 8601). updated_at string No Timestamp when the user was last updated. status string No User status (e.g., active, inactive).
